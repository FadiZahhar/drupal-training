Twig Template Files

Files end in .html.twig
  • Supply the markup that renders what is seen in browsers
  • Contain mix of HTML 5 syntax and Twig syntax

Add Custom CSS

ATTENTION

Drupal has default regions that it uses if you do not declare any custom regions in .info.yml
Declare ONE custom region, you need to declare all regions.

Twig templates render from most specific to least specific

field.html.twig
node.html.twig
region.html.twig
page.html.twig
html.html.twig
There are a lot more than these visit this site for more information

Some of the main twig Templates usages

html.html.twig
  • Declares head and body
  • HTML and Twig syntax
  • Declares document type
page.html.twig
  • Implement regions
  • Add div containers/classes
  • Declares document type
